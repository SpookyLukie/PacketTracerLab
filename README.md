<h1>Building a Network using Packet Tracer</h1>

<h2>Objective</h2>
The objective of this project is to practice building network topologies and to familiarize myself with the logical and physical visualization of networks. 
<br />


<h3>Languages and Utilities Used</h3>

- <b>Command Prompt</b>

<h3>Skills Developed</h3>

- How to structure a network between multiple devices and establishing connectivity between different computers
